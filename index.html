A limited, collectable, and deflationary asset, Monke Token is a BEP20 token written in Solidity. A set 1% of each transaction is automatically added to the liquidity pool on PancakeSwapV2, 1% of each transaction will automatically be distributed to the token holders

Want to become a Monke Token holder?

1. Download MetaMask.

2. Open the MetaMask extension and connect to the Binance Smart Chain.

3. Select "Add Token" on the bottom of the extension and paste the Monke Token contract address (listed below)

Congratulations! You are now ready to trade $MONK on PancakeSwap
